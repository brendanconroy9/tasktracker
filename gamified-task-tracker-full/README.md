This is a ready-to-run starter repository for a mobile task-tracking app with gamification and a rewards system.

## Quick start

1. Install Expo CLI: `npm install -g expo-cli`
2. `npm install`
3. Create a Firebase project and copy your config into `src/firebase.js`
4. `npm run start` (or `expo start`)

## What is included

- React Native (Expo)
- Firebase Auth + Firestore (modular SDK)
- Gamification logic: XP, Levels, Coins, Streaks
- Simple Reward Store
- Notifications setup (Expo)
- Zustand for local state
- Tests for gamification logic (Jest)
